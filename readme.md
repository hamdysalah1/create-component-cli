# Installation

clone the repo then do the following

```
cd into repo folder `create-component-cli-main`
npm link
```

# Create Component

run the following command to create component

```
hamdy@createComp COMPONENT_NAME
```

###### You must `cd` to your component folder first

# Available Options

- Class Component (WIP)
